Isotope can hide and show item elements via the filter option. filter accepts a jQuery selector. Items that match that selector will be shown. Items that do not match will be hidden.

Selectors

The simplest way to filter items is with selectors, like classes. Each item element can have several identifying classes: transition, metal, lanthanoid, alkali, etc.

``` html
...
...
...
...
...
...
...
...
```

Set a selector with the filter option. Items that match the selector will be shown. Items that do not match will be hidden. To filter .metal items:

``` js $container.isotope({ filter: '.metal' }); ```
  • .alkali, .alkaline-earth will show .alkali and .alkaline-earth item elements.
  • .metal.transition will show item elements that have both .metal and .transition classes.

jQuery selectors

If jQuery is present, Isotope uses jQuery to filter items. You can filter items with jQuery selectors. For example:

  • .metal:not(.transition) will show .metal item elements that are NOT .transition.

Functions

You can filter items with functions. This allows you to filter with a bit more logic.

``` html

80

52

20

75

...
```

If you use jQuery, you can filter with functions in jQuery

``` js $container.isotope({ // filter element with numbers greater than 60 filter: function() { // `this` is the item element. Get text of element's .number var number = $(this).find('.number').text(); // return true to show, false to hide return parseInt( number, 10 ) > 60; } }) ```

You can still filter with functions if you don’t use jQuery.

``` js iso.arrange({ // item element provided as argument filter: function( itemElem ) { var number = itemElem.querySelector('.number').innerText; return parseInt( number, 10 ) > 60; } }); ```
